Positioning

Introduction

With CSS we can position our content and elements as we please. It is just a matter of finding exactly how you want things to move, while keeping in mind how the rest of the page may react to this. For example, moving something to the center may accidently shift something else on the page down.

First lets adjust our margins, by editing our margins we ensur that no other elementt will get too close to this one


 p {
     margin-top: 100px;
     margin-bottom: 100px;
     margin-right: 150px;
     margin-left: 80px;
}

Now let's adjust our elemnts position within the border, by adjusting the padding


p {
    padding-top: 50px;
    padding-right: 30px;
    padding-bottom: 50px;
    padding-left: 80px;
}

We can also move our element blocks to our liking using float


ul {
    float: right;
    margin: 0 0 10px 10px;
}
